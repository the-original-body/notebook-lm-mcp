# Arthrose Blueprint - Request Tracker

## Naming Convention

### Request IDs
- **RQ-XXXX** - Fortlaufende Request-Nummer (z.B. RQ-0001, RQ-0002)

### Asset Types
- **RP** - Report
- **SD** - Slide Deck
- **AO** - Audio Overview
- **VO** - Video Overview
- **IF** - Infographic

### Version
- **vX** - Version (v1, v2, v3...)

### Format
```
RQ-XXXX__[TYPE]-XXvX - Titel
```

**Beispiele:**
- `RQ-0001__RP-01v1 - Die Kunst der Mäßigung`
- `RQ-0006__SD-01v1 - Arthrose Blueprint Startkurs`
